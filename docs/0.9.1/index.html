Evennia Documentation¶

This is the manual of Evennia, the open source Python MU* creation system. You can Search the documentation as well as browse all pages alphabetically in the Index. If you have trouble with unclear documentation, please let us know on our mailing list, over IRC or by dropping a note in our quick no-registration online suggestion box!
